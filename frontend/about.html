About This Project

The Event Attendance System is a full-stack web application designed for event organizers to seamlessly manage registrations and track live attendance. It solves the problem of manual, paper-based check-ins by digitizing the process through QR codes and real-time dashboard tracking.

It is built specifically for clubs, institutions, and community events that need an efficient, scalable way to handle large volumes of attendees without long queues.

Core Mechanics

QR Code Generation

During registration, the backend instantly converts participant details into a unique, scannable QR code ensuring fast and secure verification.

Live Check-in Scanner

The attendance portal utilizes device cameras via `jsQR` to scan tickets at the door, minimizing hardware dependencies for organizers.

Real-Time Dashboard

Powered by Socket.IO, the admin dashboard updates instantaneously as attendees are checked in, providing live analytics without needing a page refresh.

Tech Stack

This project is powered by a robust and lightweight stack:

Vanilla HTML/CSS/JS (Frontend) Node.js Express.js SQLite (Database) Socket.IO (Real-time updates)
