## 🚀 Overview

This repository contains the completed task for NSCC. It is structured as a full-stack application, ensuring scalable database modeling, secure backend APIs, and an interactive frontend interface.

### ✨ Features
- **Frontend App:** Responsive user interface built to specifications.
- **Backend API:** Dedicated server logic handling business rules and requests.
- **Database:** Persistent storage layer for all required application data.

## 📂 Project Structure

```
├── frontend/          # UI components and client logic
├── backend/           # Server, routing, and API controllers
└── database/          # Database configuration and schemas
```

## 🛠️ Getting Started

### Prerequisites
Make sure you have Node.js and a compatible database engine installed locally depending on the specific technologies configured in the respective folders.

### Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/SakshamJindal04/nscc_task.git
   cd nscc_task
   ```

2. **Backend Setup:**
   Navigate into the `backend` directory, install necessary dependencies (`npm install`), configure your environment variables, and start the server.

3. **Frontend Setup:**
   Navigate into the `frontend` directory, install dependencies (`npm install`), and start the development server (`npm start` or `npm run dev`).
